# NapasVPN

Десктопный VPN-клиент для Windows на протоколе **VLESS + Reality**.  
Вставь ключ → нажми Connect → готово. Системный прокси выставляется автоматически.

---

## Запуск на Windows

### Шаг 1 — Скачай xray-core

Перейди на страницу релизов:  
**https://github.com/XTLS/Xray-core/releases/latest**

Скачай файл `Xray-windows-64.zip`, распакуй архив.

> При первом запуске приложение **само скачает xray** автоматически.  
> Если автозагрузка не сработала — положи `xray.exe` вручную в папку `assets/`.

---

### Шаг 2 — Установи Node.js

Скачай и установи **Node.js 18+**:  
https://nodejs.org/en/download

---

### Шаг 3 — Установи зависимости

Открой папку с проектом в терминале (cmd или PowerShell) и выполни:

```
npm install
```

---

### Шаг 4 — Запусти приложение

```
npm start
```

Откроется окно NapasVPN.

---

### Сборка в .exe установщик

Чтобы собрать готовый установщик:

```
npm run build:win
```

Файл `NapasVPN Setup 1.0.0.exe` появится в папке `dist/`.

---

## Как пользоваться

1. Вставь VLESS-ключ в поле (формат: `vless://...`)
2. Нажми **CONNECT**
3. Системный HTTP-прокси Windows выставится автоматически
4. Для отключения нажми **DISCONNECT**

---

| Протокол   | Адрес              |

| SOCKS5     | 127.0.0.1:**10808** |
